Please (Qǐng) do this
=====================

|

`:information_source:`

   **Qǐng (请)**: please

|

| Qǐng **zuò** (请坐).

   Please sit down.
   
| Qǐng **shuō** (请说).

   Please say.
   
| Qǐng **jìn** (请进).

   Please come in.
   
| Qǐng **wèn** (请问).

   Please ask.

|
